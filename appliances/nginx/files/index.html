Quick Start

Add your site configurations to /etc/nginx/conf.d/:

incus file push mysite.conf my-nginx/etc/nginx/conf.d/
incus exec my-nginx -- nginx -s reload

Useful Commands

  • Test config: incus exec my-nginx -- nginx -t
  • Reload: incus exec my-nginx -- nginx -s reload
  • View logs: incus exec my-nginx -- tail -f /var/log/nginx/access.log
  • Health check: incus exec my-nginx -- curl -s localhost/health

Configuration Locations

  • Main config: /etc/nginx/nginx.conf
  • Site configs: /etc/nginx/conf.d/
  • Logs: /var/log/nginx/
  • Web root: /usr/share/nginx/html/
